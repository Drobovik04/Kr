Краткое описание

Сайт состоит из двух функциональных вкладок - "Художники" и "Картины". Вкладка "Художники" содержит в себе карточки для каждого художника, позволяет отфильтровать их по ФИО, также реализована функция клика по художнику, которая открывает картины, которые он написал. Вкладка "Картины" содержит в себе карточки с картинами и их названиями, позволяет фильтровать их по тегам и названию.
